# Instructions LLM (Source de vérité)

Ce document est la source de vérité pour les assistants IA utilisés dans ce dépôt.

## Objectif

Produire et maintenir du contenu pédagogique clair, exact et cohérent pour le cours 5N6 mobile avancé.

## Contexte pédagogique

- Public: étudiants de 5e session en technique informatique de gestion au CÉGEP.
- Acquis: Kotlin/Jetpack Compose, Java Spring Boot, React + WebAPI, C#, MVC, notions avancées web (dont SignalR).
- Cible principale des activités: Android.

## Règles éditoriales

- Langue: français.
- Registre: vouvoiement partout.
- Ton: direct, professionnel, pédagogique.
- Corriger les fautes (orthographe, grammaire, typographie) sans complexifier inutilement les textes.
- Préserver les choix pédagogiques de l'enseignant et éviter les réécritures massives sans demande explicite.

## Règles de contenu (docs de cours)

- Favoriser des objectifs observables et concrets dans les activités.
- Garder une terminologie Flutter précise et cohérente.
- Éviter les ambiguïtés sur la plateforme cible (Android par défaut, mention explicite si autre cible).
- En cas de doute technique sur une syntaxe Flutter/Dart, vérifier avant de corriger.

## Règles d'architecture des consignes LLM

- Ne pas dupliquer ces règles dans d'autres fichiers d'instructions.
- Les fichiers spécifiques à un assistant doivent référencer ce document.
- Si une règle locale contredit ce document, ce document prévaut, sauf indication explicite de l'utilisateur.
