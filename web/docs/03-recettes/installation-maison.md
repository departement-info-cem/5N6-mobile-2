# Installation à la maison 🏠

## Objectif

Obtenir un environnement de développement fonctionnel chez soi

## Installation de Flutter

- Accédez à la [page d'installation de Flutter](https://docs.flutter.dev/install/quick)
- Suivez les instructions d'installation pour votre distribution

:::tip
Nous recommandons l'installation via Visual Studio Code, mais si vous considérez tester Flutter avec Android Studio, vous pouvez procéder à l'installation manuelle.
:::

## Validations après installation

1. Exécutez la commande `flutter doctor` qui permet de confirmer la validité de votre environnement de développement. Vous devriez voir une liste des plateformes supportées par Flutter, et leur status. S'il y a des problèmes, vous les verrez ici. Si tout semble correct pour la plateforme Android, passez à l'étape 2.
2. Créez un projet de test et exécutez le pour s'assurer que tout fonctionne bien.

:::tip
Prenez l'habitude d'exécuter cette commande lorsque vous arrivez dans un nouvel environnement, et que quelque chose fonctionne bizzarement.
:::
