:::note Général

<Row>

<Column>

## Objectif

Implémenter l'interface graphique de votre application en Flutter, **sans back-end**.
Les données peuvent être codées en dur ou stockées en mémoire.

## Navigation

&#8203;<Highlight color="tip">4 points</Highlight> L'application comporte plusieurs écrans reliés par une navigation fonctionnelle.

- Au minimum **4 écrans** différents correspondant à vos wireframes du TP1
- La navigation entre les écrans fonctionne (push / pop)
- Un mécanisme de navigation principale est présent (Drawer, BottomNavigationBar ou équivalent)

</Column>

<Column>

## Écrans et composants

&#8203;<Highlight color="tip">6 points</Highlight> Les écrans sont implémentés avec soin.

- Chaque écran correspond à un wireframe de votre TP1
- Les éléments graphiques sont présents et bien disposés
- Les listes (si présentes) utilisent un `ListView.builder`
- La mise en page est cohérente (pas de débordement)

</Column>

</Row>

<Row>

<Column>

## Multilingue

&#8203;<Highlight color="tip">4 points</Highlight> L'application supporte le **français** et l'**anglais**.

- Toutes les chaînes de texte visibles par l'utilisateur sont externalisées dans les fichiers ARB
- Le changement de langue fonctionne (s'adapte à la langue du téléphone ou via un sélecteur)

</Column>

<Column>

## Image Picker (optionnel)

&#8203;<Highlight color="tip">2 points (conditionnel)</Highlight> Si votre application permet la sélection d'images par l'utilisateur :

- L'utilisateur peut sélectionner une image depuis la galerie du téléphone
- L'image est affichée dans l'application

Ces 2 points sont disponibles uniquement si l'image picker est pertinent pour votre application.

</Column>

</Row>

## Fonctionnement global

&#8203;<Highlight color="danger">4 points négatifs</Highlight> Votre application doit fonctionner sans plantage et correctement.
Ce pointage fonctionne en négatif.

- Plantage de l'application → **-1 point**
- Interface illisible ou débordements → **-1 point**
- Navigation brisée → **-1 point**
- Autre cas problématique → **-1 point**
